# AE4L

AE4L stands for Aerospace Engineering For Life containing aerospace learning materials adapted from [mahran-sayed/awesome-aerospace-engineering](https://github/com/mahran-sayed/awesome-aerospace-engineering/) but with up to date content and all the relevant areas aerospace engineers need to have a knowledge about. This part of the code on the website and any document linked in this Markdown file comes under the GNU Public License (GPL) V3.0 as given in the original source provided by [mahran-sayed](https://github.com/mahran-sayed/)
